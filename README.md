# aspnetcore-recipe-app
一个基于**ASP.NET Core 8**的食谱管理应用，支持用户管理、查看食谱。
# 使用技术
- ASP.NET Core 8
- SQLite
- EF Core
# 功能特性
- 支持用户注册、登录
- 创建、管理自己的食谱
- 浏览他人创建的食谱
